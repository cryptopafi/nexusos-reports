Stack-ability Analysis

Principle: EU state aid stacks across separate legal entities operating in different countries, IF each entity has an independent project. The Delaware C-Corp parent + multiple EU subsidiaries model enables this.

Italy Smart&Start (R3 anchor) + R7 programs

  • Italy SRL + Spanish SL: STACKABLE — separate entities, separate projects, separate countries
  • Italy + Portugal Lda: STACKABLE — same logic
  • Italy + Luxembourg F4S: STACKABLE — different entities, F4S is equity-free

R6 anchors (WBSO + AWS Preseed) + R7 programs

  • WBSO (NL R&D tax credit): STACKABLE with everything — different instrument type (tax credit ≠ grant)
  • AWS Preseed (Austria): STACKABLE — Austrian entity, separate project
  • WBSO + CDTI Neotec: STACKABLE — Dutch BV + Spanish SL, separate projects
De minimis alert: If any single EU subsidiary receives multiple grants all classified as "de minimis," cumulative cap is €300K per entity over 3 fiscal years. CDTI Neotec (€325K) may push above de minimis — it's typically under GBER (General Block Exemption), which has higher cumulation ceiling. GBER R&D intensity cap for SMEs: up to 80% of eligible project costs for industrial research.

Practical conclusion: Italy SRL + Spanish SL + Portuguese Lda + Dutch BV + Austrian GmbH = 5 separate entities, each eligible for their country's programs independently, without double-dipping on the same project costs.

Sources: EC State Aid overview • speedinvest non-dilutive guide
