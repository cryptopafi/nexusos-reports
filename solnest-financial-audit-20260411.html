Three-Layer Audit Stack

For any pitch deck financial audit, run all three layers in sequence.

Layer 1

Structural

Skill(audit) --profile pitch-deck --lens both

Scores narrative, source attribution, slide architecture, ask clarity, compliance hygiene against KPMG 12-slide template.

Layer 2

Methodology

F-005 + F-046 + F-094

Scores scenario diversity, unit economics completeness, moat articulation, comparables benchmarking.

Layer 3

Arithmetic

Row-by-row inline Python

Catches internal math errors the structural layer does not score. Re-derives every cell from stated assumptions.

No single layer is sufficient. All three together produce a complete picture. The audit-pro profile is necessary but not sufficient. The library procedures are necessary but not sufficient. The manual arithmetic pass is necessary but not sufficient.
