Executive Summary

  • Claude Design is real — Anthropic Labs confirmed a visual design product (Pro/Max/Team/Enterprise) that handles marketing collateral, landing pages, and social assets with brand-system integration. It exports to HTML/Canva/PDF but delegates HTML5 code generation to Claude Code. Use Claude Design for concept and visual direction; Claude Code for the actual .html/.css/.js output.
  • The best 2026 solo workflow is a 4-tool chain: Claude (copy variants) → gpt-image-2 via ChatGPT/Codex (hero visual, ~$0.05/image) → Claude Code (HTML5 layout, ~$0.015/banner) → ChatGPT Canvas (layout iteration/QA). Total cost per banner: $0.07–$0.12. Time per banner: 5–15 minutes with iteration.
  • Claude Code beats Codex for HTML/CSS design quality — confirmed by Reddit r/PPC practitioners ("Claude is better at design") and a YouTube course instructor who switched to Claude inside Codex's terminal for front-end work. Codex adds value for the gpt-image-2 native integration and background agent mode.
  • Affiliate compliance is non-negotiable and AI copy fails it by default — iGaming requires platform authorization, age gating, responsible gambling links, and geo-licensed targeting. Nutra requires FTC-substantiated claims. Sweepstakes require "No Purchase Necessary" disclosure. AI-generated copy routinely omits these; every banner needs a compliance pass before upload.
  • File weight limits are firm: 150 KB static/HTML5 initial load (Google Ads, confirmed). Standard display CTR is 0.05–0.1%; rich media/interactive reaches 0.15–0.3%. DCO (Dynamic Creative Optimization) campaigns report 20–60% higher CTR. For 20 variants/week, a Claude Pro subscription ($20/mo) covers API token costs; total budget including gpt-image-2 images: ~$5–$7/week.

3
Owner: Claude (claude.ai or API)
Copy Variants
Generate 5–10 copy variants per format: headline (≤30 chars), subheadline (≤60 chars), CTA (≤15 chars), disclaimer text. Use compliance-aware prompt (see Template A below). Compliance pass: check for prohibited language (guaranteed wins, disease claims, missing disclosures).
4
Owner: ChatGPT GPT-image-2 (or Codex)
Hero Visual Generation
Generate hero image at 1024x1024 (medium quality, ~$0.05). Use 8-frame coherence for variant sets. Scale down to banner dimensions via CSS. Text rendering in image: usable for headlines, not for dense legal copy (render that via HTML/CSS instead).
5
Owner: Claude Code
HTML5 Banner Scaffold
Claude Code generates full index.html + style.css + app.js for each size. Enforces 150 KB limit. Includes fallback static PNG. Adds click-tag parameter. Animation: CSS keyframes (no JS animation library = smaller file). Verify: du -sh banner/ + headless Chrome render.
6
Owner: ChatGPT Canvas
Layout Iteration + QA
Paste HTML into Canvas. Adjust spacing, animation timing, font size. Canvas preview pane renders live. Version history for A/B test variants. Human review: verify compliance disclosures visible, CTA tap target ≥44px on mobile, file stays under 150 KB.
7
Owner: Human
Compliance Pass + Export
Final check against network policy (platform authorization, age gate, responsible gambling link for iGaming; "No Purchase Necessary" for sweepstakes; FTC disclaimer for nutra). Zip project folder. Upload to ad server (Google Display <150 KB, Trade Desk <100 KB primary HTML).

3. SMS-Ads Specific Constraints

Affiliate Compliance by Vertical

iGaming / Sports Betting
Platform authorization required (Meta, Google, X each have separate approval processes). Age gate mandatory (18+ US; 21+ UK). Responsible gambling helpline link on landing page AND creative. Cannot target unlicensed GEOs — restrict at link level, not just ad targeting. No "guaranteed wins", no implication of skill guarantees. UK affiliates: ASA guidelines require prominent responsible gambling messaging. March 2026: Google now evaluates entire account Policy Health Score — one violation can revoke certification.
Nutraceutical / Dietary Supplements
FTC: all claims must be substantiated with competent and reliable scientific evidence. No disease treatment claims (FDA jurisdiction). Disclaimers required: "*These statements have not been evaluated by the FDA. This product is not intended to diagnose, treat, cure, or prevent any disease." Testimonials must reflect typical results or disclose atypicality.
Sweepstakes
"No Purchase Necessary" must appear prominently. Complete official rules hosted on landing page. State registration required in Florida, New York, Rhode Island for sweepstakes with prizes over $5,000. No language implying purchase improves odds.
Dating / Adult
Suggestive but non-explicit imagery on compliant networks. No implied minors. CTA must be clear about service type. Age-gating required. Affiliate must be registered formal affiliate (Meta requirement, July 2025 update).
